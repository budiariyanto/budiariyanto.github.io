Object-Oriented Design & Patterns by Cay Horstmann

Apr 19th, 2010

After searching for a book about object oriented, finally I found a very interesting ebook. The title is Object Oriented Design & Patterns by Cay Horstmann. In this book, there are only a few explanation about object oriented rules in programming, but there are many explanation and knowledge about how to design an application that implement the object oriented concept and also about popular design patterns.
This book use java programming language. As we know the java programming language is a mature language with many features. So I think it’s the best choice to help us understand about object oriented design and patterns. You can download the source code of the sample programs here.
