Slackware 13.0 Released!

Aug 29th, 2009

Akhirnya setelah sekian lama Slackware 13.0 dirilis juga. Perubahan dari Slackware 12.2 cukup banyak, maka disarankan sekali bagi para pengguna untuk mengupgrade Slackware 12.2 ke 13.0. Adapun perubahannya adalah sebagai berikut :
  • KDE 4.2.4
  • Koleksi paket-paket X yang telah disempurnakan. Sekarang tidak perlu menggunakan file konfigurasi untuk X (xorg.conf).
  • Xfce 4.6.1
  • Paket yang terkompresi lebih baik menggunakan .txz
  • Paket-paket networking, library, dan aplikasi-aplikasi untuk X yang juga diupgrade.
  • Untuk informasi lengkapnya silakan membaca official announcement dan release notes-nya.
Selamat menikmati Slackware 13.0 ini. :D
